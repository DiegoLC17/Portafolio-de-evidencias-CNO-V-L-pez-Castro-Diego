Introducción

En el área de seguridad informática, el control del tráfico de red es fundamental para proteger sistemas y evitar accesos no autorizados. En esta actividad se aplicaron reglas de iptables con el objetivo de filtrar y administrar el flujo de paquetes en una red, permitiendo únicamente servicios específicos como DNS, correo y navegación web. De esta forma, se demuestra cómo una configuración adecuada de firewall puede fortalecer la seguridad de una infraestructura y reducir riesgos ante ataques externos.

Descripción del ejercicio

Teniendo en cuenta la topología de red mostrada completa la tabla con las reglas de iptables que deberían aplicarse en el Firewall para llevar a cabo las acciones solicitadas. Las reglas, siempre que sea posible, deben determinar protocolo, dirección IP origen y destino, puerto/s origen y destino y el estado de la conexión.

Figura 1. Topología

Reglas de iptables solicitadas

1) Establecer una política restrictiva.

iptables -P DROP

2) Permitir el tráfico de conexiones ya establecidas.

iptables -A FORWARD -m state --state ESTABLISHED -j ACCEPT

3) Aceptar tráfico DNS (TCP) saliente de la red local.

iptables -A FORWARD -p tcp -d 192.1.2.10 --dport 53 -j ACCEPT

4) Aceptar correo entrante proveniente de Internet en el servidor de correo.

iptables -A POSTROUTING -p tcp -s 192.1.2.11 --dport 25 -j ACCEPT

5) Permitir correo saliente a Internet desde el servidor de correo.

iptables -A PREROUTING -p tcp -d 192.1.2.10 --dport 25 -j ACCEPT

6) Aceptar conexiones HTTP desde Internet a nuestro servidor web.

iptables -A FORWARD -p tcp -d 192.1.2.10 --dport 80 -j ACCEPT

7) Permitir tráfico HTTP desde la red local a Internet.

iptables -A FORWARD -p tcp -s 192.1.2.11 --dport 443 -j ACCEPT

Conclusión

En esta actividad se aplicaron reglas básicas de iptables para restringir el tráfico de red y permitir únicamente las conexiones necesarias. Además, se reforzó el uso de cadenas como FORWARD y el uso de estados de conexión como ESTABLISHED para garantizar un control más seguro del tráfico.

PRESENTACIÓN DEL PORTAFOLIO

El presente portafolio tiene la finalidad de concentrar y presentar de forma ordenada, profesional y clara, las evidencias académicas desarrolladas en el curso de CNO V – Seguridad Informática, de forma que puedan ser consultadas por el profesor y ser revisadas en tiempo y forma.
El contenido del portafolio se enfoca en los principales temas abordados durante el curso, tales como la seguridad informática, el análisis de vulnerabilidades y las pruebas de penetración, los cuales permiten comprender los riesgos asociados a los sistemas de información y las estrategias utilizadas para su protección.
Asimismo, este portafolio constituye una evidencia del proceso de aprendizaje, ya que refleja el desarrollo progresivo de conocimientos teóricos y habilidades prácticas adquiridas durante el curso, así como la aplicación de principios de documentación técnica y buenas prácticas en el ámbito de la seguridad informática.

PERFIL DEL ESTUDIANTE

Soy Diego López Castro, estudiante de Ingeniería en Tecnologías de la Información (I.T.I.) en la Universidad Politécnica de San Luis Potosí. Actualmente curso el octavo semestre, siendo mis principales áreas de interés académico la programación, la seguridad informática y la ciberseguridad.
Mis intereses se centran en la protección de datos e información, el análisis de vulnerabilidades, la prevención de ataques y la investigación en temas relacionados con amenazas cibernéticas. En los últimos seis meses he participado en prácticas de pruebas de penetración, cadenas de custodia y redacción de reportes técnicos vinculados con la ciberseguridad.

LENGUAJES Y TECNOLOGÍAS

El sitio web fue desarrollado utilizando HTML5 para la estructura del contenido, CSS3 para el diseño y la adaptación responsiva, y JavaScript para la implementación de funcionalidades dinámicas. Además, se integró la API EmailJS para la gestión del envío de correos electrónicos desde el formulario de contacto sin requerir un servidor propio.

PLATAFORMA DE PUBLICACIÓN

El portafolio se encuentra alojado en GitHub Pages, plataforma que permite la publicación de sitios web estáticos directamente desde un repositorio. Esta solución facilita el acceso público al proyecto, garantiza disponibilidad continua y permite integrar de forma directa el despliegue con el control de versiones del código fuente.

CONTROL DE VERSIONES

El proyecto es gestionado mediante un sistema de control de versiones basado en Git y GitHub, lo que permite registrar y organizar de forma ordenada los cambios realizados en el código fuente. Este enfoque facilita la trazabilidad de las modificaciones, el mantenimiento del proyecto y la recuperación de versiones estables ante posibles errores o ajustes durante el desarrollo.

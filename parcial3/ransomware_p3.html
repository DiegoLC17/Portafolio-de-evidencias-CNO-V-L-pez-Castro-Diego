¿Qué es un ransomware?

Es un tipo de malware que está diseñado para mantener cautivo un sistema computacional o los datos que contiene hasta que se realice un pago. El ransomware generalmente funciona cifrando sus datos para que no pueda acceder a ellos.
Algunas otras versiones de ransomware pueden aprovechar vulnerabilidades específicas del sistema para bloquearlo. El ransomware a menudo se propaga a través de correos electrónicos de phishing que lo alientan a descargar un archivo adjunto malicioso o a través de una vulnerabilidad.

03. Validación del entorno

Muchos códigos maliciosos revisan si están siendo analizados o si el entorno no les conviene. 


        def check_enviroment():
          Imports Microsoft.Win32

          Module StartupModule
            Sub SetStartup(ByVal enable As Boolean)
              ' Abrir la llave del registro para el usuario actual
              Dim rk As RegistryKey = Registry.CurrentUser.OpenSubKey("SOFTWARE\Microsoft\Windows\CurrentVersion\Run", True)

              If enable Then
                ' "MiAppVB" es el nombre que verá el sistema
                ' Application.ExecutablePath obtiene la ruta del .exe actual
                rk.SetValue("MiAppVB", Application.ExecutablePath)
              Else
                ' Eliminar la entrada si el usuario decide desactivarlo
                rk.DeleteValue("MiAppVB", False)
              End If
            End Sub
          End Module
